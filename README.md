# University-cpp-Labs

> Every line committed. Every concept documented. This is what learning C++ seriously looks like.

---

## About

This repository contains all lab submissions for **Programming Fundamentals** — a first-semester CS course at NUML University, Islamabad. Rather than treating labs as throwaway assignments, every program here is version-controlled, documented, and written with intent.

The goal: build a foundation in C++ that goes deeper than the syllabus.

---

## What's covered

Labs follow a strict progression — no topic touched until the previous one is solid:

| Milestone | Topics |
|-----------|--------|
| ✅ Data Types & Variables | Primitive types, memory sizes, type casting |
| ✅ Loops | for, while, do-while, nested loops |
| ✅ Operators | Arithmetic, logical, bitwise |
| ✅ Functions | Parameters, pass by value/reference, overloading, scope |
| ⬜ Arrays & Strings | In progress |
| ⬜ Pointers & Memory | Upcoming |
| ⬜ File I/O | Upcoming |
| ⬜ OOP Basics | Upcoming |

---

## Why this repo exists

Most students submit lab work and move on. This repo exists because:

- Code without version control is code that disappears
- Employers want to see how you think, not just what you submitted
- Building the habit of clean, documented code starts on day one — not after graduation
